# Leorio Clinic Manager (Mobile App)

## Concept

A mobile-first clinic management application built for Leorio to manage:

- Patients
- Appointments
- Treatments

The app is designed to run entirely on mobile without requiring a desktop or web system.

## Features

- Add new patients
- Schedule appointments
- Track treatments
- Delete patients
- Local data persistence using AsyncStorage

## Tech Stack

- React Native (Expo)
- AsyncStorage

## How to Run

1. Install dependencies:

   npm install

2. Start Expo:

   npx expo start

3. Scan QR code using Expo Go app.

The app will run directly on your phone.
